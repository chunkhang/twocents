# Two Cents

> A todo list for couples

## Dependencies

- [Docker](https://www.docker.com/)
- [Go](https://golang.org/)
- [Postgres](https://www.postgresql.org/)

## Building

- `docker-compose build`

## Running

- `docker-compose up -d`
- `open http://localhost`

## Logging

- `docker-compose logs -f app`

## Stopping

- `docker-compose down`
